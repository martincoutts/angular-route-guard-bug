Route Guard Bug

To replicate the bug:
  1. Make sure you are on the homepage (/home)
  2. Navigate to the object component using link on homepage
  3. Navigate back using one of the "Navigate Back" buttons
  4. The URL will change to home and then an alert from the route guard will be displayed
  5. The URL will be "/home" and when you dismiss the alert the URL will switch back to "/object"
The navigation using location.back or history.back occurs and the URL changes before the route guard can intervene and prevent the URL changing.
